# Solution

Evidence-based technical solution advisor for AI agents. Multi-source cross-validated research across 10+ channels to help you make informed technology decisions.

## Install

```bash
npx skills add qq919006380/solution
```

## What It Does

When you need to choose between technologies, libraries, or frameworks, this skill:

1. Asks targeted questions to understand your context
2. Dispatches parallel research agents across 10 channels:
   - GitHub, npm/PyPI, Context7 official docs
   - Stack Overflow, Reddit, Hacker News, Lobste.rs
   - Chinese communities (Juejin, Zhihu, V2EX)
   - Benchmark data, Security databases (Snyk/CVE)
   - Google search for recent articles
3. Cross-validates findings across sources
4. Delivers an ASCII comparison table with scored recommendations

## Example Triggers

- "Which state management library should I use?"
- "React vs Vue vs Svelte for my project"
- "Recommend a database for high-write workloads"
- "/solution"
